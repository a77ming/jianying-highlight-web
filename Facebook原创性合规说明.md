# 🎯 Facebook原创性合规 - 完整指南

本项目已全面升级，确保AI生成的所有剪辑方案都符合Facebook的"有意义润色"（Meaningful Commentary）标准，避免被判定为缺乏原创性。

---

## 📋 Facebook原创性政策解读

### ❌ 违规行为（绝对禁止）

1. **简单剪辑拼接**：只是将片段按时间顺序拼接
2. **仅添加视觉装饰**：水印、徽标、边框等
3. **仅添加背景音乐**：音乐不构成原创性
4. **调整播放速度**：加速/减速视频
5. **添加相同语言字幕**：仅翻译或转录对话
6. **简单介绍**："这个视频讲的是..."这种无价值介绍
7. **反应视频但价值极小**：只是看视频反应，无附加分析
8. **汇总但无润色**：收集多个来源但无创意加工
9. **重复发布**：已发布内容重新发布无改动

### ✅ 合规行为（必须做到）

#### 1. 独特的画外音/旁白 ⭐⭐⭐（最重要）

**要求**：
- ❌ 不能只是翻译或复述对话
- ✅ 必须提供独特观点、分析或解读

**合格示例**：
- "这是一个典型的操控现场。注意他的用词..."
- "如果你遇到这种情况，请记住..."
- "这背后反映的是心理学上的..."

**不合格示例**：
- "他说：我爱你"
- "她在哭"

#### 2. 创意性剪辑手法

**合格手法**：
- 倒叙悬疑法（先结果再原因）
- 对比揭示法（表象vs真相）
- 非线性叙事（插叙、多线）
- 悬念设置（问题引导）
- 情感递进（从平静到激烈）

#### 3. 信息字幕方案

**必须添加**：
- 人物标签（操控者？OR 保护者？）
- 心理活动（她不知道、他犹豫了）
- 关系说明（前男友、上司、敌人）
- 关键词强调（放大、变色、闪烁）
- 数据或图示

#### 4. 新信息内容

**必须提供**：
- 背景知识补充
- 人物心理分析
- 剧情隐喻解读
- 社会议题探讨
- 教育性内容（心理学、社会学等）
- 幽默吐槽或讽刺评论

## 🎬 本项目的合规保证

### AI提示词系统

本项目使用专门的提示词系统（`lib/prompts.ts`），确保AI生成的每个方案都包含：

#### 必须包含的元素（≥3项）：

A. **独特画外音/旁白**（必须）
   - 不是简单翻译或复述
   - 提供新视角、新解读、新分析
   - 有明确观点或情感立场

B. **创意性剪辑手法**（至少2种）
   - 非线性叙事结构
   - 对比、悬念、情感递进等

C. **视觉增强元素**（至少1种）
   - 信息字幕（人物标签、心理活动、关系说明）
   - 强调文字或符号
   - 特效说明

D. **新信息内容**（至少1种）
   - 背景知识、分析解读、教育意义等

### 原创性自检清单

AI生成方案时会自动检查：
- ✓ 画外音提供了独特观点？
- ✓ 剪辑有创意结构（非简单拼接）？
- ✓ 添加了新信息？
- ✓ 视觉上有创意元素？
- ✓ 整体叙事有明确立场？
- ✓ 内容无法仅靠观看原视频获得？

## 🎓 常见画外音风格

本项目AI会使用以下风格之一：

### 1. 侦探解读风
- 像侦探分析案件一样分析人物动机
- "注意他的用词...这暗示..."
- 适合：悬疑、推理、复杂剧情

### 2. 情感共鸣风
- 从情感角度切入，引发观众共鸣
- "如果你是她，你会怎么做？"
- 适合：爱情、家庭、情感剧

### 3. 幽默吐槽风
- 幽默吐槽剧中的不合理或戏剧化桥段
- "这种霸道总裁戏码真的还存在？"
- 适合：偶像剧、玛丽苏、狗血剧情

### 4. 教育科普风
- 科普相关知识（心理学、社会学等）
- "这在心理学上称为'煤气灯效应'..."
- 适合：涉及专业知识的剧情

### 5. 社会议题风
- 联系现实社会议题进行讨论
- "这反映了当代社会的..."
- 适合：有社会深度的剧情

## 📝 画外音脚本示例

### 优秀的画外音（符合标准）

```
「这是一个教科书级的操控现场。表面上看，Nate是一个陌生人
为了保护Brooke而承诺撒谎。但注意他的用词——'Rookie mistake'
（新手错误）。这暗示他不是第一次处理尸体。更关键的是，
他为什么要对一个刚认识的人承诺这一切？真正的保护者会报警，
而不是替你埋尸。这种'保护'背后，往往隐藏着更危险的控制欲。
如果你遇到这种'完美保护者'，请记住：最快愿意为你犯罪的人，
往往也是最容易对你犯罪的人。」
```

**为什么合格**：
- ✓ 提供了独特分析（操控现场）
- ✓ 有教育价值（识别操控）
- ✓ 有明确观点和建议
- ✓ 无法仅靠观看原视频获得

### 不合格的画外音

```
「Nate对Brooke说，他会报警。他说他会说自己发现了尸体。
Brooke问他愿意为一个陌生人撒谎吗？然后Nate拥抱她说
一切都会好的。」
```

**为什么不合格**：
- ✗ 只是复述对话
- ✗ 没有任何分析或观点
- ✗ 可以通过看原视频获得
